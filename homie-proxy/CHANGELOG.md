# Changelog

## 1.0.0 — Initial release

- WebSocket proxy with server-side token injection
- Serves Homie dashboard at port 3001 + HA ingress
- LAN/WAN auto-detection with 1.5 s timeout
- Rate limiting: 10 connections/min per IP
- Service call audit log
- Multi-connection support (multi-property / multi-tenant)
- Supports aarch64, amd64, armhf, armv7, i386
